# Appendix G VS2005 Team Suite Performance Profile

This appendix provides the best known method for using Microsoft Visual Studio
2005\* Team Suite to get performance data. It involves "porting" the EFI code to
a Win32 console application and then using the VS Performance Wizard to figure
out how to tune it.
